이것은 네트워크와 장치의 독립성이 필요하며, 이는 사용자가 다양한 접속 네트워크를 이동하거나 컴퓨팅 장치를 변경할 수 있도록 허용한다
Seamless services
끊김없는 서비스

It is responsible for supporting seamless services across heterogeneous networks that require connection migration from one network to another
이것은 한 네트워크에서 다른 네트워크로 연결을 이동해야 하는 이종 네트워크 간 끊김 없는 서비스를 지원하는 역할을 한다.

heterogeneous networks (HetNets)
이종 네트워크

In a heterogeneous environment, It represents the basis for providing continuous network connectivity to mobile users roaming between access networks
이종 환경에서, 이것은 네트워크 간 로밍하는 모바일 사용자에게 지속적인 네트워크 연결을 제공하기 위한 기초를 나타낸다
It affects the whole protocol stack, from the physical, data link, and network layers up to transport and application layers
이것은 물리 계층, 데이터 링크 계층, 네트워크 계층에서 전송 계층 및 응용 계층까지 프로토콜 스택 전체에 영향을 미친다.
mobility management

Mobility support
이동성 지원

It is the ability of a MT to move between IP subnets, while continuing to be reachable for incoming requests and maintaining sessions across subnet changes
이것은 단말(MT)이 IP 서브넷 간 이동하면서도 들어오는 요청에 응답할 수 있고, 서브넷 변경 간 세션을 유지할 수 있는 능력이다.
Terminal Mobility
단말 이동성

It concentrates on the movement of users instead of users terminals and involves the provision of personal communications and personalized operating environments
이것은 사용자 단말 대신 사용자의 이동에 초점을 맞추며, 개인화된 통신과 개인화된 작업 환경 제공을 포함한다.
Personal mobility
개인 이동성

It allows users to maintain a session while chagning terminals
이것은 사용자가 단말을 변경하면서도 세션을 유지할 수 있도록 한다.
Session mobility
세션 이동성

it can defined as the ability of users to maintain access to their services even moving and changing tetminals or servies providers
이것은 사용자가 이동하거나 단말 또는 서비스 제공자를 변경해도 자신의 서비스에 접근을 유지할 수 있는 능력으로 정의된다.
Service mobility
서비스 이동성

it is a mechanism used to configure one computer with more than one network interface and multiple IP address
이것은 한 컴퓨터를 여러 네트워크 인터페이스 및 다중 IP 주소로 설정하는 매커니즘이다
It provides enhanced and reliable Internet connectivity without compromising efficient performance.
이것은 효율적인 성능을 손상시키지 않으면서도 향상된 신뢰성과 인터넷 연결을 제공한다.
Multihoming
다중 접속

It related to the periodical updating of the new MN’s location on the access link to allow the network to keep track of the MN.
네트워크가 단말(MN)의 위치를 추적할 수 있도록 새로운 단말 위치를 접속 링크에서 주기적으로 갱신하는 것
Location registration/update
위치 등록/업데이트

It is ability of the network to find the MN’s current location in the access network in order to deliver its data packets.
네트워크가 단말(MN)의 현재 위치를 찾아 데이터 패킷을 전달할 수 있도록 하는 능력을 의미
Traffic delivery
트래픽 전달

It is the process by which an MT keeps its connection active when it moves from one PoA to another.
단말(MT)이 하나의 접속점(PoA)에서 다른 접속점으로 이동할 때 연결을 유지하는 과정
Handover management
핸드오버 관리

Mobility and multihoming try to solve the same problem related to the session survivability in two different environments and by using different mechanisms.
이동성과 다중 접속은 두 가지 다른 환경에서 각각 다른 메커니즘을 사용하여 세션 지속성과 관련된 동일한 문제를 해결하려고 함

It operate from different layers of the protocol stack with the goal of minimizing performance degradation during the handover procedure
이것은 핸드오버 절차 동안 성능 저하를 최소화하는 것을 목표로 하여 프로토콜 스택의 다양한 계층에서 작동한다.
